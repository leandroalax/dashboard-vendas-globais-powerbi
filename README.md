# Global Sales Dashboard – Power BI

## Business Context
This project presents an analytical dashboard developed in Microsoft Power BI with the objective of analyzing **global sales performance**, supporting decision-making related to revenue, product categories, discounts, and geographic distribution.

The dashboard was created as part of the *Microsoft Power BI for Business Intelligence and Data Science* program at Data Science Academy and follows a business-driven analytical approach.

Site: https://www.datascienceacademy.com.br/

---

## Business Questions Addressed
The dashboard was designed to answer the following key business questions:

1. What is the total sales value?
2. How many sales were made by product category?
3. How many sales were made by country considering delivery priority?
4. What is the average discount applied by product subcategory?
5. Which countries have the highest average sales value? (Geographic visualization)

Additionally, the dashboard allows dynamic filtering by:
- Year
- Customer Segment
- Country

## Data Limitations
Although suitable for analytical and educational purposes, the dataset presents some limitations when compared to real-world corporate data:
- No missing values
- Highly standardized categorical fields
- Absence of extreme outliers
- Data structure already optimized for analysis

These characteristics were considered during analysis and documentation.
